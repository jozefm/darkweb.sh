Installation & Setup

INSTALLATION

  1. Install very basic Debian, Debian download, (Small or Tiny) Remember Less is More!
  2. Warning: if you are running Kali Linux Kali Download or ParrotOS Download there are a lot of unnecessary tools and services running on these distros that may not be ideal for a Tor Investigative System. If you do use these systems, make sure you shut them down after use.  📴
  3. Install ClamAV | sudo apt install clamav | sudo clamscan 🐚
  4. Install UFW Uncomplicated firewall 🧱| apt install ufw | systemctl status ufw
  5. Next we will install requirements for our VPN Connectivity 🔦 with OpenVPN and it dependencies
    • sudo apt install -y openvpn network-manager-openvpn network-manager-openvpn-gnome

   6. Once you finish your base Operating system installation – You will use a specialized script we created to help download other required tools:

    1. Next run our setup script: curl -sSL /darksetup.sh/darksetup.sh | sudo bash
    2. Make executable chmod +x darksetup.sh
    3. Run executable ./darksetup.sh   (wait for tools/install to finish)
  2. Next download our container management script called “darkweb.sh“
  3. Container Manager: darkweb.sh right click and save to your system
  4. You may need to make it executable on your distro -> chmod +x darkweb.sh

  5. Next, Install docker onto your system, after you have Docker installed “service docker start” & “service docker status” [if not click here “docker install” on Kali ]

To use the container manager “darkstart.sh” 🔐script to start the various containers use the following syntax

  1. switch to the darkweb directory [root\darkweb or user\darkweb]
  2. sudo ./darkweb.sh -h provides a help menu
  3. sudo ./darkweb.sh start alpine ‘starts the alpine container’ server we will use this for hosting our Dark Web Server on the Tor Network.
  4. There are multiple containers that are downloaded you can see everything that is available docker -ls

SETUP TOR: A Step by Step Setup:

 

 First we will install Tor via the command line so open a terminal and type in the following:

$ sudo apt update  (updates our list of repositories and any items that needs to be updated)
$ sudo apt install tor torbrowser-launcher

This installs “Tor” as a daemon, and the “Tor browser”

What is a Daemon

Tor can be run as a standalone daemon or as a browser app, hence the two installations.

Verify tor the daemon is installed via the command line. so in the same open terminal type in the following:

$ tor –version

Tor version 0.4.7.10

If tor is not started you can start it by the following command:

$ service tor start  (sudo service tor start) service tor status,  service tor stop

or:

$ systemctl start/stop/status tor

Note: However this is not necessary to just run the Tor Browser,

Next we can verify that the tor daemon is working using the ipify.org site, a simple public ip address api

$ wget -qO – https://api.ipify.org; echo

$ ip_address

next we will use torsocks

torify (now just a wrapper for torsocks)

$ torsocks wget -qO – https://api.ipify.org; echo

$ tor ip_address

Note: Torsocks can “torify” your command line entries by default, I recommend against this but show it here for completeness of the service and capabilities.

$ source torsocks on

Tor mode activated. Every command will be torified for this shell.  and of course to stop this shell interaction, just use the “off” command

torrc -located at /etc/tor/torrc

We recommend you never mess with this file unless done so at the instructions of a trusted advisor. Errors or certain entries can compromise and open your system up to attack on the tor network.

Tired of remote’ing into machines and seeing small text, small search and small tabs? The solution is to create and edit your userChrome.css |  The operation is similar on both linux and windows.

userChrome.css

To create your own custom userChrome.css there are a few programs you can use.

Here is a subreddit that discusses these changes:

How do I change the font size in the address bar/menu bar/tool bar? Or possibly change to a better font?
byu/ombx infirefox

 

 

The Manual way:

To Zoom the whole browser. This will change more than the tabs, but it does increase font sizes.

  1. Open Firefox and type in about:config

  2. Search for layout.css.devPixelsPerPx

  3. Change from -1.0 to 1.75
 ———————————————–

To change the font size only of specific parts with in the browser. This will be a bit more work, the below command creates larger text on the tabs.

  1. Open about:config

  2. Search for toolkit.legacyUserProfileCustomizations.stylesheets and toggle it to true.

  3. Go to about:profiles

  4. Find your profile’s Root Directory and click Open Directory.

  5. In your profile directory, create a folder called chrome

  6. Open the chrome directory, and create a text file named userChrome.css

  7. to increase the font size on the TABS, Put the text below into the userChrome.css file

.tab-text.tab-label{
    font-size: 20px;
}

———————————————–

To change the default text size from 12px in the search area past this code into the userChrome.css below the tab size font-size

Select and copy the following style rule code

/*** Larger text in toolbar area ***/
#navigator-toolbox {
   /* 125% of Windows default of 12px => 15px */
  font-size: 20px !important;
}
